# Simple Resume Builder

A web-based application to create professional resumes easily.

## Features

- Interactive form to input personal information, experience, education, skills, projects, and certifications
- Live preview of the resume as you type
- Dark mode toggle for better user experience
- Print-friendly design for saving as PDF
- Responsive layout for mobile and desktop

## How to Use

1. Open `index.html` in your web browser
2. Fill out the form sections with your information
3. View the live preview on the right
4. Click "Generate Resume" to see the final output
5. Use "Print / Save as PDF" to download your resume

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
